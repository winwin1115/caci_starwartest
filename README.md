# Test-Project(CACI)

# Spec Requirement

- Next.js
- Typescript
- Tailwind CSS
- Redux, Redux Tool Kit

# Pre-requirement

- Install Node.js 18.2.0
- Clone the repository

On the terminal, command like below

```bash
git clone https://github.com/winwin1115/caci_starwartest.git
```

- Install node module

```bash
yarn install
```

- Run Next.js server

```bash
yarn dev
```

# Project Structure

- assets

  - Contain all assets like svg, png...

- components

  - Button, Header, Input, Loading, SelectBox component...

- hooks

  - All customized hooks functions..

- layout
  - Two layouts, authentication layout(landing page) and main layout (main page).
- modules
  - custom hooks
- pages
  - main web pages.
- services
  - RTK service
- store
  - store in redux.
- styles
  - contain css files.
- types
  - define typescript interfaces and types

## Build Pipeline Command

```bash
yarn build
```
